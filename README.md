# Mayja E-commerce

Tienda online para Mayja con catalogo de productos (Hogar, Juguetes Sensoriales, Tecnologia, Ropa), carrito de compras, pago contra entrega, y panel de administracion propio.

## Stack Tecnologico
- Frontend: React + TailwindCSS
- Backend: Node.js + Express
- Base de datos: PostgreSQL
- Almacenamiento de imagenes: Cloudinary (gratuito)
- Control de versiones: Git + GitHub
- Servidor: Digital Ocean Droplet
- Deployment: PM2 + Nginx
